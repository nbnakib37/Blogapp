First, extract the ZIP file, then move the project folder outside OneDrive, for example to C:\AndroidProjects\SimpleBlogApp; after that open Android Studio, click Open, select the SimpleBlogApp folder, wait for Gradle Sync to finish, connect your phone with USB, enable Developer options and USB debugging, choose File Transfer/MTP, press Allow USB debugging on the phone, then in Android Studio select your phone from the device dropdown and click the green Run ▶ button. If you get a build error, close Android Studio, delete the app\build folder, reopen the project, click Build → Clean Project, and run again.
