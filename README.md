# Simple Blog App

A beginner-friendly Android Studio project made for the module task list.

## What the app includes

- Multiple views / screens:
  - MainActivity: shows all blog posts in a ListView
  - AddPostActivity: lets the user add a new post
  - DetailActivity: shows the full blog post
- Components:
  - Button
  - EditText
  - Switch
  - TextView
  - ImageView
  - ListView
- Android concepts:
  - Activities
  - Intents
  - BroadcastReceiver
  - IntentService
  - Custom ListView layout

## How to run

1. Open Android Studio.
2. Choose **Open**.
3. Select the `SimpleBlogApp` folder.
4. Wait for Gradle sync to finish.
5. Press **Run**.

## Beginner note

The app stores posts only while the app is running. It does not use a database, because the purpose is to keep the project simple and suitable for beginner Android practice.

## Learning diary reflection example

In this project, I created a simple blog application in Android Studio. I practiced creating multiple Activities and moving between screens using Intents. I used a ListView with a custom layout to display blog posts, including a title, category, preview text, and image. I also added input components such as EditText, Button, and Switch so the user can create a new blog post. The project helped me understand how Android screens, UI components, and simple event handling work together in a mobile application.
